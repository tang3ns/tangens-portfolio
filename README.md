# tangens-portfolio
# 📐 TANGENS.DESIGN | Digital Portfolio

Официальный сайт дизайн-студии Платона. Фокус на эстетике, системном подходе и автоматизации бизнес-процессов.

## 🚀 О проекте
Сайт-портфолио, созданный для демонстрации навыков в UX/UI дизайне и технической реализации современных веб-интерфейсов. 

**Основные разделы:**
* **Hero**: Визуальная концепция и позиционирование студии.
* **Method**: Описание рабочего процесса от идеи до реализации.
* **Work**: Избранные кейсы: лендинги, интерфейсы и мобильные версии.

## 🛠 Технологический стек
* **Frontend**: HTML5, CSS3 (Modern Grid/Flexbox), JavaScript.
* **Design**: Figma, Framer.
* **AI Tools**: v0.dev, Cursor (AI Code Editor).
* **Deploy**: Vercel.
